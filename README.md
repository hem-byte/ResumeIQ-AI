# ResumeIQ AI

AI-powered ATS Resume Analyzer built using Flask and Google Gemini

## Features

- Resume PDF Upload
- ATS Score Analysis
- Strength Detection
- Missing Skill Detection
- Improvement Suggestions
- Final Verdict Dashboard

## Technologies

- Python
- Flask
- Google Gemini API
- PyPDF2
- HTML
- CSS
- Langchain
  

## Installation

pip install flask google-generativeai PyPDF2

python app.py
